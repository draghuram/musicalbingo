Creating new game

  • Prepare a CD with the list of songs you want to play. Preferably in MP3 format. You may consider normalizing the MP3 songs so that all songs have similar volume levels.
  • Prepare a text file like titles.txt with this list of songs. Each line should have one title. Make sure titles are not too long. Otherwise, they wrap around in chits. The file should have the extension ".txt". Click here for example file. You may also make a copy of this file, name it like titles.m3u and place it on the CD. This will be useful in playing the songs on computer.
  • Upload titles.txt here - Generate Bingo Chits - to get chits. Follow the instructions on this page and print the chits.
  • There will be 3 chits per page. Cut them into individual chits. If you have lot of pages, you may consider going to a place like Kinkos to get them cut.
  • Now, play the songs in random order. You can use any MP3 player for this purpose like a DVD player, Real Player, or Windows Media Player. If you are playing on a computer, you can play the m3u file created above which will be recognized by most players as a playlist. Don't forget to select random play option.
  • You can get a master list here - Generate Master List. This list can be used to keep track of songs already played.
  • Give one chit to each player, play the songs, and enjoy!
